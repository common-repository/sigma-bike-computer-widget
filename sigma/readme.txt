This widget generates a Sigma BC906 bike computer showing your bike statistics.

== Description ==

This widget uses php-gd to generate a Sigma BC906 bike computer to show your bike statistics. The statistics can be entered by using the dashboard.

== Installation ==

Extract the zip archive in your WordPress plugins folder (wp-content/plugins) and then activate the plugin by using the WordPress dashboard. The activation sets up initial configuration options and creates the database table for the plugin data.
